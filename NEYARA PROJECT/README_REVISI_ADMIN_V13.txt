REVISI ADMIN UX V13 - SPA KAMPOENG SUKOHARJO

Fokus revisi:
1. Dashboard admin diringkas agar informasi tidak terlalu padat.
2. Pendapatan lunas ditonjolkan sebagai metrik utama.
3. Grafik pendapatan bulanan memiliki pilihan rentang 3 bulan, 6 bulan, 12 bulan, dan semua data.
4. Format tanggal tampilan dibuat konsisten menjadi DD/MM/YYYY.
5. Riwayat booking menampilkan informasi waktu data dibuat dan diperbarui.
6. Menu sidebar "CRUD Booking" dan "CRUD Layanan" diganti menjadi "Booking" dan "Layanan".
7. Form booking admin menghapus input nama belakang agar sesuai dengan form user yang memakai nama lengkap.
8. Tabel booking dilengkapi filter status, filter tanggal, sortir, pilihan jumlah tampilan 10/100/1000, dan tombol View More.
9. Aksi simpan, hapus, ubah status, pulihkan default, dan hapus semua data memakai pop-up konfirmasi.
10. Form layanan dilengkapi kuota per hari, batas informasi, penghitung karakter deskripsi, dan penghitung jumlah manfaat/informasi.
11. Kolom tabel booking diperbesar dan dibuat lebih stabil agar data penting lebih mudah dibaca.

Catatan:
Website ini masih memakai localStorage. Untuk implementasi produksi, data booking dan pendapatan sebaiknya dipindahkan ke backend/database agar aman dan tidak terbuka di browser pengguna.
