Perubahan:
1. Font seluruh website diperbesar secara global.
2. Body text dinaikkan ke kisaran 18-21 px pada desktop.
3. Label kecil dinaikkan ke kisaran 16-17 px agar tidak sulit dibaca.
4. Tombol dan input dinaikkan ke kisaran 17-21 px dengan tinggi minimal 54 px.

6. Halaman utama, admin, payment, success, dan admin redirect ikut diperbarui.

Lakukan hard refresh dengan Ctrl + F5 setelah membuka website agar browser tidak memakai cache CSS lama.
