✦ User Manual

Panduan Booking dan Bantuan Pelanggan

Panduan ini ditambahkan agar website lebih sesuai untuk prototype, dokumentasi penggunaan, dan kebutuhan pelanggan yang baru pertama kali melakukan booking online.

📌

Cara Booking

Pilih layanan, isi nama, WhatsApp, tanggal, jam, lalu tekan tombol lanjut pembayaran.

Data apa saja yang wajib diisi?

Nama lengkap, nomor WhatsApp, tanggal booking, jam booking, dan pilihan layanan wajib diisi. Email bersifat opsional.

💳

Pembayaran

Website mendukung Virtual Account, QRIS, dompet digital, dan pembayaran di tempat sesuai kebutuhan pelanggan.

Apa yang harus dilakukan setelah membayar?

Simpan ID booking. Jika perlu, pelanggan dapat menghubungi admin melalui tombol WhatsApp dengan menyertakan ID booking.

🛟

Troubleshooting

Jika halaman tidak berpindah, periksa kembali kolom wajib, nomor WhatsApp, dan pilihan layanan.

Bagaimana jika salah jadwal?

Hubungi admin melalui WhatsApp dan sebutkan ID booking agar admin dapat membantu pengecekan data.
